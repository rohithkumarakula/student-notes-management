# 📚 Student Notes Management System

A Flask-based web application that allows students to securely upload, manage, search, download, and delete PDF notes.

## 🚀 Features

- User Registration
- Secure Login & Logout
- Password Hashing
- Session Management
- Upload PDF Notes
- Download PDF Notes
- Search Notes
- Delete Notes
- Dashboard with Total Notes
- Multi-user Support (Each user sees only their own notes)

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5
- Bootstrap 5
- Jinja2

## ▶️ How to Run

```bash
pip install -r requirements.txt
python database.py
python app.py
```

Open:

```
http://127.0.0.1:5000
```
